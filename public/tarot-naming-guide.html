5. 自动重命名脚本

如果您有大量图片需要重命名，可以使用以下脚本自动处理：

Windows PowerShell脚本：

# 创建映射表 $cardMapping = @{ "愚人" = "fool.jpg" "魔术师" = "magician.jpg" "女祭司" = "high-priestess.jpg" "女皇" = "empress.jpg" "皇帝" = "emperor.jpg" "教皇" = "hierophant.jpg" "恋人" = "lovers.jpg" "战车" = "chariot.jpg" "力量" = "strength.jpg" "隐士" = "hermit.jpg" "命运之轮" = "wheel-of-fortune.jpg" "正义" = "justice.jpg" "倒吊人" = "hanged-man.jpg" "死神" = "death.jpg" "节制" = "temperance.jpg" "恶魔" = "devil.jpg" "高塔" = "tower.jpg" "星星" = "star.jpg" "月亮" = "moon.jpg" "太阳" = "sun.jpg" "审判" = "judgement.jpg" "世界" = "world.jpg" } # 设置图片目录 $imageDir = "C:\Path\To\Your\Images" # 遍历目录中的图片文件 Get-ChildItem -Path $imageDir -Filter "*.jpg" | ForEach-Object { $fileName = $_.Name $baseName = $_.BaseName # 检查是否需要重命名 if ($cardMapping.ContainsKey($baseName)) { $newName = $cardMapping[$baseName] $newPath = Join-Path -Path $imageDir -ChildPath $newName # 执行重命名 Write-Host "重命名: $fileName -> $newName" Rename-Item -Path $_.FullName -NewName $newName } } Write-Host "重命名完成！"

使用说明：

  1. 将脚本保存为 rename-tarot.ps1
  2. 修改 $imageDir 为您的图片目录路径
  3. 在PowerShell中运行脚本：.\rename-tarot.ps1

6. 验证图片路径

修改完成后，可以通过以下方式验证图片路径是否正确：

  • 在浏览器中访问：http://localhost:3000/tarot-cards/major-arcana/magician.jpg
  • 如果能看到魔术师的图片，则路径正确
  • 如果显示404错误，请检查文件名和路径是否正确
